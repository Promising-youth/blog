方法
====

rust中的struct没有继承的概念，其实现继承的方式类似golang中的组合，但是golang中的组合第一个字段如果不加名字，就是默认继承，但是rust中不是这种方式。rust中实现类似的组合继承方式是通过deref.即`impl Deref for xxx{}` 

如代码:
```rust
struct Foo{}
trait Object{
	fn test();
}

impl Object for Foo{
	fn test(){
	}
}

struct FooSon{
	foo:Foo,
}

impl Deref for FooSon {
    type Target = Foo;

    fn deref<'a>(&'a self) -> &'a Node {
        &self.foo
    }
}
```
